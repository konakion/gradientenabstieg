# Gradientenabstieg am Beispiel neuronaler Netze

Dieses Repository enthält die Folien zu der **Vorlesung „Gradientenabstieg am Beispiel neuronaler Netze“** im Kontext von **Machine Learning und Künstlicher Intelligenz**.

Die Präsentation vermittelt Schritt für Schritt ein **intuitives und mathematisches Verständnis dafür, wie neuronale Netze lernen**, indem ihre Parameter mithilfe von Optimierungsverfahren angepasst werden.

## Inhalt der Präsentation

Die Folien behandeln unter anderem folgende Themen:

- Motivation von Optimierungsproblemen in Statistik, Data Science und Machine Learning  
- Einführung in neuronale Netze (Weights, Bias, Aktivierungsfunktion)  
- Beispiel aus der Elektrotechnik: Sensorsignal und Klassifikationsproblem  
- Loss-Funktion und Modellfehler  
- Bedeutung des Gradienten und Zusammenhang mit Ableitungen  
- Gradientenabstieg als iterativer Optimierungsalgorithmus  
- Schritt-für-Schritt-Beispiel zur Minimierung der Residuenquadratsumme  
- Rolle der Lernrate beim Training  
- Erweiterung zu Stochastic Gradient Descent (SGD)

## Zielgruppe

Die Vorlesung richtet sich an Studierende der **Elektrotechnik, Informatik oder verwandter Studiengänge** mit grundlegenden Kenntnissen in:

- Analysis (Ableitungen)
- Linearer Regression
- Grundlagen des Machine Learning
